## 使用方式  

当前最新版本为：
[![](https://jitpack.io/v/kymjs/Common.svg)](https://jitpack.io/#kymjs/Common)

```
// root build.gradle
allprojects {
    repositories {
        maven { url 'https://jitpack.io' }
    }
}

// module build.gradle
dependencies {
    implementation 'com.github.kymjs:common:2.0.7'
}
```
